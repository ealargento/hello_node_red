hello-world
===========

Testing node-red

### About

This is a basic app developed with the sole purpose of getting to know node-red. It connects to an API from weatherapi.com and shows the values of temperature through a dashboard for 3 different cities: Tampere, Kangasala and (my home town in Venezuela) Maracay.
